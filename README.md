### Setting up the game
`$ git clone https://github.com/MisteryDush/5x5.git` <br>
`$ cd 5x5` <br>
`$ fpc 5x5` <br>
`$ 5x5`

### Controls

w - Move up <br>
s - Move down <br>
a - Move left <br>
d - Move right <br>
space - Select cell <br>
r - Restart the game <br>
esc - Quit <br>
